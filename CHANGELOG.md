# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of AsyncAPI Generator
- Support for generating AsyncAPI 2.4.0 specifications from Go code
- AST-based type extraction for accurate schema generation
- Support for all AsyncAPI protocols (NATS, AMQP, MQTT, Kafka, etc.)
- Parameterized channels with `{param}` syntax
- NATS wildcard subscriptions (`*` and `>`)
- Request-reply pattern support
- Comprehensive documentation with examples
- NATS example application
- CI/CD pipelines with GitHub Actions
- Automated releases
- Code coverage reporting
- golangci-lint integration
- Dependabot for dependency updates

### Documentation
- Complete README with badges
- AsyncAPI annotations reference
- NATS subject patterns guide
- Contributing guidelines
- Code of conduct
- Examples and quick start guide
